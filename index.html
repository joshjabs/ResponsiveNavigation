Responsive Navigation Menus

Navigation Menus are typically built using an Unordered List < ul > of List Items < li > that contain Hyperlinks < a >


    <ul>
        <li>
            <a href="#">Menu Item 1</a>
        </li>
        <li>
            <a href="#">Menu Item 2</a>
        </li>
    </ul>

Apply Different CSS Styles Without Affecting Your HTML

Harness CSS to apply full stylizations to your menu

                
    nav .links {
        float: right;
        padding: 0;
        margin: 0;
        width: 60%;
        height:100%;
        display: flex;
        justify-content: space-around;
        align-items: center;
    }


Import High Quality Google Fonts from CDN

Browse thousands of easy-to-import fonts at Google Fonts

                  
    @import url('https://fonts.googleapis.com/css2?family=Roboto&display=swap');

    * {
        font-family: 'Anton', sans-serif;
    }

Use CSS @media Rule to change stylizations based on device size

Responsively display different menu styles for mobile screens

                
    @media screen and (max-width: 768px) {
        nav .logo {
            padding-left:0px;
            float: none;
            width: auto;
            justify-content: center;
        }
        nav .links {
            float: none;
            position: fixed;
            z-index:9;
            flex-direction: column;
            height: auto;
            width: auto;
            left: 0;
            right: 0;
            top: 100px;
            bottom: 100%;
            background-color: rgba(0,0,0,0.8);
            overflow:hidden;
            transition: all .5s ease-in-out;
        }
    }
